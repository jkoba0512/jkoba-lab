# jkoba-lab

Source for **Jun Kobayashi's academic site** at
https://jkoba0512.github.io/jkoba-lab/.

Kyushu Institute of Technology · Dept. of Intelligent and
Control Systems · Associate Professor.
Research: biologically-inspired robotics, liquid neural
networks, active inference, musculoskeletal modeling.

## Built with

- [Hugo](https://gohugo.io/) static site generator
- [HugoBlox academic-cv](https://github.com/HugoBlox/hugo-theme-academic-cv) template
- Deployed via GitHub Actions to GitHub Pages

## Local development

```bash
hugo server -D
```

Visit http://localhost:1313 to preview.
